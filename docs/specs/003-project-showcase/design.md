# Design: Project Showcase

## Card Layout

Each project card should include:

- Project name
- Short one-line description
- Tech stack tags
- Problem
- What I built
- Security/operations details
- What I learned
- GitHub/documentation link if available

## Visual Style

Use terminal-style status blocks:

service: navidrome
status: running
access: tailscale-only
security: no-public-ports

For Home Lab:

host: raspberry-pi
ssh: key-based
containers: docker-compose
reverse-proxy: nginx
monitoring: enabled
