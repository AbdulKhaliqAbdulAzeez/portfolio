# Spec 003: Project Showcase

## Goal

Show Abdul's strongest technical work through polished project cards.

## Projects

1. Raspberry Pi Music Server
2. Home Lab Raspberry Pi Server

## User Stories

### US-006: Visitor can understand the Raspberry Pi Music Server project

As a recruiter, I want to see what Abdul built, what tools he used, and why it matters.

#### Acceptance Criteria

- Project card lists Navidrome, Tailscale, and Linux
- Project explains self-hosted music streaming
- Project explains secure remote access through Tailscale
- Project explains that public port exposure was avoided

### US-007: Visitor can understand the Home Lab Server project

As a recruiter, I want to see Abdul's Linux, Docker, and systems administration skills.

#### Acceptance Criteria

- Project card mentions key-based SSH
- Project card mentions disabled password/root login
- Project card mentions Docker Compose
- Project card mentions Nginx reverse proxy
- Project card mentions Portainer
- Project card mentions backups and monitoring
