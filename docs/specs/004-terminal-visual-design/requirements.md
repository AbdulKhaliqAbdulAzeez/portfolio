# Spec 004: Terminal Visual Design

## Goal

Create a portfolio style inspired by a terminal/system dashboard without overwhelming the visitor.

## User Stories

### US-008: Visitor experiences a technical visual identity

As a visitor, I want the portfolio to visually communicate IT, Linux, and systems work.

#### Acceptance Criteria

- Site uses a dark theme
- Hero includes terminal-inspired text
- Sections use command-line-style labels
- Visual effects are subtle and do not hurt readability

### US-009: Visitor is not distracted by unnecessary extras

As a recruiter, I want the site to remain easy to read and professional.

#### Acceptance Criteria

- No music player
- No sound effects
- No heavy animations
- No unnecessary language switcher
- No cluttered widgets
