# Design: Terminal Visual Design

## Theme

Background:
- Near-black or dark navy

Text:
- White/off-white for main text
- Muted gray for secondary text

Accent:
- Green, cyan, or amber

Typography:
- Monospace for terminal labels
- Sans-serif for readable body text

## Hero Example

abdul@portfolio:~$ ./start
Initializing profile...
Status: Available for IT / support / infrastructure opportunities
Location: Jersey City, NJ

## Animation Rules

Allowed:
- Soft fade-in
- Cursor blink
- Small hover transitions

Avoid:
- Long typing animations
- Audio
- Excessive glitch effects
- Animations that delay reading
