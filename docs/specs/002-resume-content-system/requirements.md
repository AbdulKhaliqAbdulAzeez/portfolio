# Spec 002: Resume Content System

## Goal

Turn Abdul's resume into structured portfolio content that can be reused across sections.

## Source Content

Use resume details for:

- Name
- Email
- LinkedIn
- GitHub
- Location
- Summary
- Education
- Certifications
- Projects
- Skills

## User Stories

### US-004: Visitor can read Abdul's professional summary

As a recruiter, I want to read a short summary so I can understand Abdul's background.

#### Acceptance Criteria

- Summary mentions Information Technology student at NJIT
- Summary mentions Linux, networking, systems support, Python, SQL, and home lab work
- Summary is shorter and more conversational than the resume version

### US-005: Visitor can view education

As a recruiter, I want to see Abdul's school and expected graduation date.

#### Acceptance Criteria

- Education section lists NJIT
- Degree is B.S. in Information Technology
- Expected graduation is May 2027
- Relevant coursework is shown
