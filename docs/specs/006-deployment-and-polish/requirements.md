# Spec 006: Deployment and Polish

## Goal

Deploy the portfolio and make it ready to share with recruiters, internship applications, and LinkedIn/GitHub profiles.

## User Stories

### US-013: Abdul can share a live portfolio link

As Abdul, I want the site deployed so I can include it on my resume, GitHub, LinkedIn, and job applications.

#### Acceptance Criteria

- Site is deployed publicly
- Site loads without errors
- Contact links work
- Resume link works
- GitHub and LinkedIn links work

### US-014: Portfolio feels complete

As a visitor, I want the site to feel polished and intentional.

#### Acceptance Criteria

- No placeholder text remains
- No broken links remain
- Spacing is consistent
- Content is proofread
- Projects are clearly written
